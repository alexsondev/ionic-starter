# Mobile App Starter

## Setup

Make sure to install the dependencies:

```bash

pnpm install

```

## Development Server

Start the development server on `http://[network.ip]:3000`:

```bash
# pnpm
pnpm dev -o --host

```

Simulator

```bash
#iOS
pnpm ios

#Android
pnpm android
```
